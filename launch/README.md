# ADG Launch Kit

Launch checklist for Agent Data Gateway.

## Prerequisites
- [ ] GitHub repo is public with code pushed
- [ ] CI passing (lint, test, build badges)
- [ ] Docker image published or Docker Compose tested on clean machine

## Channels

### Hacker News (Show HN)
- Post: `hn-show.md`
- Best time: 10:00–12:00 PT / 17:00–19:00 UTC (peak HN traffic)
- Prepare to respond in comments for first 2 hours
- Have 3-5 comment responses ready (see hn-show.md)

### Reddit
- `r/LLMDevs` — primary, most relevant audience
- `r/selfhosted` — good for self-hosted + Docker angle
- `r/MCP` — protocol-specific audience
- `r/devops` — more general/ops perspective
- Best time: weekdays 7:00–9:00 UTC
- Posts: `reddit-posts.md`

### Discord
- LangChain Discord (#show-and-tell)
- MCP Discord (#tools-integrations)
- AI Agent Dev Discord (#projects)
- Messages: `discord-announcements.md`

### Twitter/X
- Thread: `twitter-thread.md`
- Tag @LangChainAI, relevant MCP accounts
- Use hashtags: #MCP #AIAgents #oss

## Post-Launch
- [ ] Monitor comments/feedback for first 24h
- [ ] Collect feature requests into ADG backlog
- [ ] Update README with FAQs from launch
- [ ] Track signups / GitHub stars / Docker pulls

## Content Strategy
- **Primary angle:** Security — "AI agents shouldn't have your passwords"
- **Secondary angle:** Simplicity — "10 lines of YAML, one docker compose up"
- **Distribution angle:** MCP — "turn any DB into MCP tools"
